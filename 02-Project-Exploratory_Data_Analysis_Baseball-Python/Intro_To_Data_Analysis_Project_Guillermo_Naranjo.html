Dataset and organization¶

This jupyter notebook will contain both project report and code. I will use The Lahman Baseball Database, working with four CSVs:

  • MASTER - Player names, DOB, and biographical info
  • Salaries - player salary data

Initial questions that I would like to ask witht the analysis¶

As an team manager I want to know how money traslates to wins to drive investment in key positions.

  • Is there a relation between salaries and wins.
  • If so, which teams have the best wins/salaries ratio. (this remains interesting to see team with good budget performance)
  • What is the distribution of player's salaries of the team with most wins in 2014 by position. I concluded that there is not a direct relation between money and wins so, I will explore other variables that translate to wins by looking a best performing teams. #Removing outliers, defining a function
def remove_outliers(dataset, field):
    Q1 = dataset[field].quantile(0.25)
    Q3 = dataset[field].quantile(0.75)    
    IQR = Q3 - Q1
    not_outliers = (dataset[field] > Q1 - 1.5 * IQR) & (dataset[field] < Q3 + 1.5 * IQR)
    return dataset[not_outliers]
In [1143]:
def isOutlier(dataset, field):
    Q1 = dataset[field].quantile(0.25)
    Q3 = dataset[field].quantile(0.75)    
    IQR = Q3 - Q1
    outliers = (dataset[field] < Q1 - 1.5 * IQR) | (dataset[field] > Q3 + 1.5 * IQR)
    return outliers
